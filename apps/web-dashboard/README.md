# web-dashboard

Frontend web cho user va admin.

Phan scaffold hien tai:

- React + Vite + TypeScript
- 1 web app cho ca user va admin
- Co san dashboard co ban de noi voi backend
- Ho tro nhan realtime qua WebSocket

Scripts:

- `npm run dev`
- `npm run build`
- `npm run preview`

Route hien tai:

- `/` cho user view
- `/admin` cho admin view
